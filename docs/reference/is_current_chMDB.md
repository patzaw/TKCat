# Check if the [chMDB](https://patzaw.github.io/TKCat/reference/chMDB.md) object refers to the current instance of the MDB

Check if the [chMDB](https://patzaw.github.io/TKCat/reference/chMDB.md)
object refers to the current instance of the MDB

## Usage

``` r
is_current_chMDB(x)
```

## Arguments

- x:

  a [chMDB](https://patzaw.github.io/TKCat/reference/chMDB.md) object

## Value

A single logical: TRUE if x refers to the current instance of the MDB.
