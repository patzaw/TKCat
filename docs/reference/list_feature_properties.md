# List properties of a feature

List properties of a feature

## Usage

``` r
list_feature_properties(kmr, feature)
```

## Arguments

- kmr:

  a [KMR](https://patzaw.github.io/TKCat/reference/create_KMR.md) object

- feature:

  the name of the feature

## Value

A [dplyr::tibble](https://dplyr.tidyverse.org/reference/reexports.html)
with the description of feature properties
