# List possible units for a type of measurement

List possible units for a type of measurement

## Usage

``` r
list_measurement_units(kmr, measurement)
```

## Arguments

- kmr:

  a [KMR](https://patzaw.github.io/TKCat/reference/create_KMR.md) object

- measurement:

  the type of measurement

## Value

A [dplyr::tibble](https://dplyr.tidyverse.org/reference/reexports.html)
with the description of supported units
