# Unarchive a chMDB in a [chTKCat](https://patzaw.github.io/TKCat/reference/chTKCat.md)

Unarchive a chMDB in a
[chTKCat](https://patzaw.github.io/TKCat/reference/chTKCat.md)

## Usage

``` r
unarchive_chMDB(x, name)
```

## Arguments

- x:

  a [chTKCat](https://patzaw.github.io/TKCat/reference/chTKCat.md)
  object

- name:

  the name of the database to archive

## Value

No return value, called for side effects
