# Check if the object is an [MDB](https://patzaw.github.io/TKCat/reference/MDB.md) object

Check if the object is an
[MDB](https://patzaw.github.io/TKCat/reference/MDB.md) object

## Usage

``` r
is.MDB(x)
```

## Arguments

- x:

  any object

## Value

A single logical: TRUE if x is an MDB object.
