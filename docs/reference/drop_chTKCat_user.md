# Drop a user from a [chTKCat](https://patzaw.github.io/TKCat/reference/chTKCat.md) object

Drop a user from a
[chTKCat](https://patzaw.github.io/TKCat/reference/chTKCat.md) object

## Usage

``` r
drop_chTKCat_user(x, login)
```

## Arguments

- x:

  a [chTKCat](https://patzaw.github.io/TKCat/reference/chTKCat.md)
  object

- login:

  login of the user to drop

## Value

No return value, called for side effects
