# Change chTKCat password

Change chTKCat password

## Usage

``` r
change_chTKCat_password(x, login, password)
```

## Arguments

- x:

  a [chTKCat](https://patzaw.github.io/TKCat/reference/chTKCat.md)
  object

- login:

  user login

- password:

  new user password

## Value

No return value, called for side effects
