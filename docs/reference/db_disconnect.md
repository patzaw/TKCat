# Disconnect an object from a database

Disconnect an object from a database

## Usage

``` r
# S3 method for class 'chMDB'
db_disconnect(x)

# S3 method for class 'chTKCat'
db_disconnect(x)

db_disconnect(x)

# S3 method for class 'metaMDB'
db_disconnect(x)
```

## Arguments

- x:

  an object with a database connection

## Value

No return value, called for side effects
