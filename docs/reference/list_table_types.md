# List types of tables defined in a [KMR](https://patzaw.github.io/TKCat/reference/create_KMR.md) object

List types of tables defined in a
[KMR](https://patzaw.github.io/TKCat/reference/create_KMR.md) object

## Usage

``` r
list_table_types(kmr)
```

## Arguments

- kmr:

  a [KMR](https://patzaw.github.io/TKCat/reference/create_KMR.md) object

## Value

A [dplyr::tibble](https://dplyr.tidyverse.org/reference/reexports.html)
with the names of table types, their descriptions and the related
collections
