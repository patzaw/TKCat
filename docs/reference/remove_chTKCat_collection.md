# Remove a collection from a [chTKCat](https://patzaw.github.io/TKCat/reference/chTKCat.md) database

Remove a collection from a
[chTKCat](https://patzaw.github.io/TKCat/reference/chTKCat.md) database

## Usage

``` r
remove_chTKCat_collection(x, title)
```

## Arguments

- x:

  a [chTKCat](https://patzaw.github.io/TKCat/reference/chTKCat.md)
  object

- title:

  the title of the collection to remove

## Value

No return value, called for side effects
