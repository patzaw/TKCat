# Check if the object is a [chMDB](https://patzaw.github.io/TKCat/reference/chMDB.md) object

Check if the object is a
[chMDB](https://patzaw.github.io/TKCat/reference/chMDB.md) object

## Usage

``` r
is.chMDB(x)
```

## Arguments

- x:

  any object

## Value

A single logical: TRUE if x is a
[chMDB](https://patzaw.github.io/TKCat/reference/chMDB.md) object
