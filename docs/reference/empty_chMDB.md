# Empty a chMDB in a [chTKCat](https://patzaw.github.io/TKCat/reference/chTKCat.md)

Empty a chMDB in a
[chTKCat](https://patzaw.github.io/TKCat/reference/chTKCat.md)

## Usage

``` r
empty_chMDB(x, name, timestamp = NA)
```

## Arguments

- x:

  a [chTKCat](https://patzaw.github.io/TKCat/reference/chTKCat.md)
  object

- name:

  the name of the database to empty

- timestamp:

  timestamp of the instance to empty. If NA (default) the current
  instance is emptied.

## Value

No return value, called for side effects
