# List available [POK](https://patzaw.github.io/TKCat/reference/create_POK.md)

List available
[POK](https://patzaw.github.io/TKCat/reference/create_POK.md)

## Usage

``` r
list_POKs(x, kmr)
```

## Arguments

- x:

  a [chTKCat](https://patzaw.github.io/TKCat/reference/chTKCat.md) or
  [TKCat](https://patzaw.github.io/TKCat/reference/TKCat.md) object

- kmr:

  [KMR](https://patzaw.github.io/TKCat/reference/create_KMR.md) object
  with KM requirements or its name in tkcat

## Value

The names of available POKs in x with kmr requirements.
