# List collections available in a [chTKCat](https://patzaw.github.io/TKCat/reference/chTKCat.md)

List collections available in a
[chTKCat](https://patzaw.github.io/TKCat/reference/chTKCat.md)

## Usage

``` r
list_chTKCat_collections(x, withJson = FALSE)
```

## Arguments

- x:

  a [chTKCat](https://patzaw.github.io/TKCat/reference/chTKCat.md)
  object

- withJson:

  if TRUE, returns the json strings of the collection (default: FALSE)

## Value

A tibble with the title, the description and optionally the json
definition of the collections
