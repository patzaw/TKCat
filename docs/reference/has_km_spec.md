# Check if KM specifications are available in an [MDB](https://patzaw.github.io/TKCat/reference/MDB.md) object

Check if KM specifications are available in an
[MDB](https://patzaw.github.io/TKCat/reference/MDB.md) object

## Usage

``` r
has_km_spec(x, kmr)
```

## Arguments

- x:

  an [MDB](https://patzaw.github.io/TKCat/reference/MDB.md) object with
  specification tables

- kmr:

  an [MDB](https://patzaw.github.io/TKCat/reference/MDB.md) object with
  KM requirements

## Value

A logical: TRUE if the MDB has KM specifications
