# Get the default mapper function for a collection

Get the default mapper function for a collection

## Usage

``` r
get_collection_mapper(collection)
```

## Arguments

- collection:

  the name of the targeted collection (it should belong to local
  collections: see
  [`list_local_collections()`](https://patzaw.github.io/TKCat/reference/list_local_collections.md)).

## Value

A function to map collection members.
