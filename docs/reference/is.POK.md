# Check if the object is a [POK](https://patzaw.github.io/TKCat/reference/create_POK.md) object

Check if the object is a
[POK](https://patzaw.github.io/TKCat/reference/create_POK.md) object

## Usage

``` r
is.POK(x)
```

## Arguments

- x:

  any object

## Value

A single logical: TRUE if x is a
[POK](https://patzaw.github.io/TKCat/reference/create_POK.md) object
