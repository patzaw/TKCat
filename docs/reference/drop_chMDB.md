# Drop a database from a [chTKCat](https://patzaw.github.io/TKCat/reference/chTKCat.md)

Drop a database from a
[chTKCat](https://patzaw.github.io/TKCat/reference/chTKCat.md)

## Usage

``` r
drop_chMDB(x, name)
```

## Arguments

- x:

  a [chTKCat](https://patzaw.github.io/TKCat/reference/chTKCat.md)
  object

- name:

  the name of the database to remove

## Value

No return value, called for side effects
