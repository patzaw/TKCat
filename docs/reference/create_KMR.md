# Create an [MDB](https://patzaw.github.io/TKCat/reference/MDB.md) object with Knowledge Management Requirements (KMR)

Create an [MDB](https://patzaw.github.io/TKCat/reference/MDB.md) object
with Knowledge Management Requirements (KMR)

## Usage

``` r
create_KMR(name, title, description, version, maintainer)
```

## Arguments

- name:

  the name of the requirements

- title:

  the title of the requirements

- description:

  the description of the requirements

- version:

  version of the requirements

- maintainer:

  maintainer of the requirements

## Value

An MDB object with KM requirements data model
