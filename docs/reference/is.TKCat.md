# Check the object is a [TKCat](https://patzaw.github.io/TKCat/reference/TKCat.md) object

Check the object is a
[TKCat](https://patzaw.github.io/TKCat/reference/TKCat.md) object

## Usage

``` r
is.TKCat(x)
```

## Arguments

- x:

  any object

## Value

A single logical: TRUE if x is a
[TKCat](https://patzaw.github.io/TKCat/reference/TKCat.md) object
