# Get a list of MDB from [metaMDB](https://patzaw.github.io/TKCat/reference/metaMDB.md) object

Get a list of MDB from
[metaMDB](https://patzaw.github.io/TKCat/reference/metaMDB.md) object

## Usage

``` r
MDBs(x)
```

## Arguments

- x:

  a [metaMDB](https://patzaw.github.io/TKCat/reference/metaMDB.md)
  object

## Value

A list of MDB objects
