# Add empty KM specification tables to an [MDB](https://patzaw.github.io/TKCat/reference/MDB.md) object

Add empty KM specification tables to an
[MDB](https://patzaw.github.io/TKCat/reference/MDB.md) object

## Usage

``` r
add_km_spec(x, kmr)
```

## Arguments

- x:

  an [MDB](https://patzaw.github.io/TKCat/reference/MDB.md) object to
  update with specification tables

- kmr:

  an [MDB](https://patzaw.github.io/TKCat/reference/MDB.md) object with
  KM requirements

## Value

An [MDB](https://patzaw.github.io/TKCat/reference/MDB.md) object with
empty KM specification tables
