Rename a TKCat object

## Usage

``` r
TKCat(..., list = NULL)

# S3 method for class 'TKCat'
names(x) <- value

# S3 method for class 'TKCat'
rename(.data, ...)

# S3 method for class 'TKCat'
x[i]

# S3 method for class 'TKCat'
c(...)
```

## Arguments

- ...:

  TKCat objects

- list:

  a list of [MDB](https://patzaw.github.io/TKCat/reference/MDB.md)
  objects

- x:

  a TKCat object

- value:

  new [MDB](https://patzaw.github.io/TKCat/reference/MDB.md) names

- .data:

  a TKCat object

- i:

  index or names of the MDB to take

## Value

a TKCat object
