# Check a [chTKCat](https://patzaw.github.io/TKCat/reference/chTKCat.md) object

Check a [chTKCat](https://patzaw.github.io/TKCat/reference/chTKCat.md)
object

## Usage

``` r
check_chTKCat(x, verbose = FALSE)
```

## Arguments

- x:

  a [chTKCat](https://patzaw.github.io/TKCat/reference/chTKCat.md)
  object

- verbose:

  a logical indicating if information messages should be displayed.

## Value

Invisible result:
[chTKCat](https://patzaw.github.io/TKCat/reference/chTKCat.md) object
