# Get KM specifications from an [MDB](https://patzaw.github.io/TKCat/reference/MDB.md) object

Get KM specifications from an
[MDB](https://patzaw.github.io/TKCat/reference/MDB.md) object

## Usage

``` r
get_km_spec(x, kmr)
```

## Arguments

- x:

  an [MDB](https://patzaw.github.io/TKCat/reference/MDB.md) object with
  specification tables

- kmr:

  an [MDB](https://patzaw.github.io/TKCat/reference/MDB.md) object with
  KM requirements

## Value

An [MDB](https://patzaw.github.io/TKCat/reference/MDB.md) object with
kmr specification tables from x
