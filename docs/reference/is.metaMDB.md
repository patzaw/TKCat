# Check if the object is a [metaMDB](https://patzaw.github.io/TKCat/reference/metaMDB.md) object

Check if the object is a
[metaMDB](https://patzaw.github.io/TKCat/reference/metaMDB.md) object

## Usage

``` r
is.metaMDB(x)
```

## Arguments

- x:

  any object

## Value

A single logical: TRUE if x is an
[metaMDB](https://patzaw.github.io/TKCat/reference/metaMDB.md) object
