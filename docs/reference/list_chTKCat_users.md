# List [chTKCat](https://patzaw.github.io/TKCat/reference/chTKCat.md) user

List [chTKCat](https://patzaw.github.io/TKCat/reference/chTKCat.md) user

## Usage

``` r
list_chTKCat_users(x)
```

## Arguments

- x:

  a [chTKCat](https://patzaw.github.io/TKCat/reference/chTKCat.md)
  object

## Value

A tibble with 3 columns:

- login: user login

- contact: user contact information

- admin: if the user is an admin of the chTKCat object
