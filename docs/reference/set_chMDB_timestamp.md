# Set timestamp of the current version of an MDB in [chTKCat](https://patzaw.github.io/TKCat/reference/chTKCat.md)

Set timestamp of the current version of an MDB in
[chTKCat](https://patzaw.github.io/TKCat/reference/chTKCat.md)

## Usage

``` r
set_chMDB_timestamp(x, name, timestamp)
```

## Arguments

- x:

  a [chTKCat](https://patzaw.github.io/TKCat/reference/chTKCat.md)
  object

- name:

  the name of the database to affect

- timestamp:

  a single POSIXct value as a timestamp for the chMDB instance.

## Value

No return value, called for side effects
