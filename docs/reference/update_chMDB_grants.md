# Update grants on tables in an MDB of a [chTKCat](https://patzaw.github.io/TKCat/reference/chTKCat.md) object

The update is done automatically based on user access.

## Usage

``` r
update_chMDB_grants(x, mdb)
```

## Arguments

- x:

  a [chTKCat](https://patzaw.github.io/TKCat/reference/chTKCat.md)
  object

- mdb:

  name of the modeled database

## Value

No return value, called for side effects
