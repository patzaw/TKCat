# Get a [KMR](https://patzaw.github.io/TKCat/reference/create_KMR.md) object from a [TKCat](https://patzaw.github.io/TKCat/reference/TKCat.md) or a [chTKCat](https://patzaw.github.io/TKCat/reference/chTKCat.md) object

Get a [KMR](https://patzaw.github.io/TKCat/reference/create_KMR.md)
object from a [TKCat](https://patzaw.github.io/TKCat/reference/TKCat.md)
or a [chTKCat](https://patzaw.github.io/TKCat/reference/chTKCat.md)
object

## Usage

``` r
get_KMR(...)
```

## Arguments

- ...:

  parameters for the
  [`get_MDB()`](https://patzaw.github.io/TKCat/reference/get_MDB.md)
  function

## Value

A [KMR](https://patzaw.github.io/TKCat/reference/create_KMR.md) object
