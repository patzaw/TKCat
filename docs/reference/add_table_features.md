# Add possible features to table type in [KMR](https://patzaw.github.io/TKCat/reference/create_KMR.md)

Add possible features to table type in
[KMR](https://patzaw.github.io/TKCat/reference/create_KMR.md)

## Usage

``` r
add_table_features(kmr, table, features)
```

## Arguments

- kmr:

  a [KMR](https://patzaw.github.io/TKCat/reference/create_KMR.md) object

- table:

  the name of the table type

- features:

  a character vector with optional features for this table type

## Value

The modified KMR [MDB](https://patzaw.github.io/TKCat/reference/MDB.md)
object
