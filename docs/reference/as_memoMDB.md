# Convert any MDB object in a [memoMDB](https://patzaw.github.io/TKCat/reference/memoMDB.md) object

Convert any MDB object in a
[memoMDB](https://patzaw.github.io/TKCat/reference/memoMDB.md) object

## Usage

``` r
as_memoMDB(x, ...)
```

## Arguments

- x:

  a MDB object

- ...:

  additional parameters for the
  [`memoMDB()`](https://patzaw.github.io/TKCat/reference/memoMDB.md)
  function.

## Value

A [memoMDB](https://patzaw.github.io/TKCat/reference/memoMDB.md) object
