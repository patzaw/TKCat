# Check the object is a [chTKCat](https://patzaw.github.io/TKCat/reference/chTKCat.md) object

Check the object is a
[chTKCat](https://patzaw.github.io/TKCat/reference/chTKCat.md) object

## Usage

``` r
is.chTKCat(x)
```

## Arguments

- x:

  any object

## Value

A single logical: TRUE if x is a
[chTKCat](https://patzaw.github.io/TKCat/reference/chTKCat.md) object
