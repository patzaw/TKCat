# Count the number of records

Count the number of records

## Usage

``` r
# S3 method for class 'MDB'
count_records(x, ...)

count_records(x, ...)
```

## Arguments

- x:

  an object with embedded data tables

- ...:

  the name of the tables to consider (default: all of them)

## Value

A named vector with the number of records per table.
