# Get a [POK](https://patzaw.github.io/TKCat/reference/create_POK.md) from a [chTKCat](https://patzaw.github.io/TKCat/reference/chTKCat.md) connection or a [TKCat](https://patzaw.github.io/TKCat/reference/TKCat.md) object

Get a [POK](https://patzaw.github.io/TKCat/reference/create_POK.md) from
a [chTKCat](https://patzaw.github.io/TKCat/reference/chTKCat.md)
connection or a
[TKCat](https://patzaw.github.io/TKCat/reference/TKCat.md) object

## Usage

``` r
get_POK(x, mdb, kmr)
```

## Arguments

- x:

  a [chTKCat](https://patzaw.github.io/TKCat/reference/chTKCat.md) or a
  [TKCat](https://patzaw.github.io/TKCat/reference/TKCat.md) object

- mdb:

  [MDB](https://patzaw.github.io/TKCat/reference/MDB.md) object with KM
  specifications or its name in tkcat

- kmr:

  [KMR](https://patzaw.github.io/TKCat/reference/create_KMR.md) object
  with KM requirements or its name in tkcat

## Value

A [POK](https://patzaw.github.io/TKCat/reference/create_POK.md) object
