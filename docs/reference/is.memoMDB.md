# Check if the object is a [memoMDB](https://patzaw.github.io/TKCat/reference/memoMDB.md) object

Check if the object is a
[memoMDB](https://patzaw.github.io/TKCat/reference/memoMDB.md) object

## Usage

``` r
is.memoMDB(x)
```

## Arguments

- x:

  any object

## Value

A single logical: TRUE if x is an
[memoMDB](https://patzaw.github.io/TKCat/reference/memoMDB.md) object
