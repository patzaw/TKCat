# Check if the object is a [fileMDB](https://patzaw.github.io/TKCat/reference/fileMDB.md) object

Check if the object is a
[fileMDB](https://patzaw.github.io/TKCat/reference/fileMDB.md) object

## Usage

``` r
is.fileMDB(x)
```

## Arguments

- x:

  any object

## Value

A single logical: TRUE if x is an
[fileMDB](https://patzaw.github.io/TKCat/reference/fileMDB.md) object
