# Add a user to an MDB of a [chTKCat](https://patzaw.github.io/TKCat/reference/chTKCat.md) object

Add a user to an MDB of a
[chTKCat](https://patzaw.github.io/TKCat/reference/chTKCat.md) object

## Usage

``` r
add_chMDB_user(x, mdb, login, admin = FALSE)
```

## Arguments

- x:

  a [chTKCat](https://patzaw.github.io/TKCat/reference/chTKCat.md)
  object

- mdb:

  name of the modeled database

- login:

  login of the user to drop

- admin:

  if the user is an admin of the MDB

## Value

No return value, called for side effects
