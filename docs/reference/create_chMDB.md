# Create a database in a [chTKCat](https://patzaw.github.io/TKCat/reference/chTKCat.md)

Create a database in a
[chTKCat](https://patzaw.github.io/TKCat/reference/chTKCat.md)

## Usage

``` r
create_chMDB(x, name, public = FALSE)
```

## Arguments

- x:

  a [chTKCat](https://patzaw.github.io/TKCat/reference/chTKCat.md)
  object

- name:

  the name of the new database

- public:

  if the database data are accessible to any user (default:FALSE)

## Value

No return value, called for side effects
