# Get the data files from a [fileMDB](https://patzaw.github.io/TKCat/reference/fileMDB.md) object

Get the data files from a
[fileMDB](https://patzaw.github.io/TKCat/reference/fileMDB.md) object

## Usage

``` r
data_files(x)
```

## Arguments

- x:

  a [fileMDB](https://patzaw.github.io/TKCat/reference/fileMDB.md)
  object

## Value

a list with "dataFiles" and "readParameters" for reading the files.
